**HR Attrition Analysis Dashboard | Power BI · SQL · Excel**

Developed an interactive HR Analytics dashboard using the IBM HR dataset (1,470 employee records) to uncover key factors influencing employee attrition. The project focuses on transforming raw HR data into actionable insights to support data-driven decision-making for workforce retention.

Performed data cleaning and transformation using Power Query and SQL techniques, followed by building a structured data model in Power BI. Designed dynamic DAX measures to calculate attrition rates across multiple dimensions such as department, job role, age group, tenure, and salary.

The dashboard presents insights through intuitive and visually engaging reports, including KPI cards, slicers, drill-through functionality, and bookmarks for seamless navigation. It enables HR professionals—even with no technical background—to easily explore trends and identify high-risk attrition segments.

**Key Insights Delivered**

Identified departments and job roles with the highest attrition rates

Analyzed the relationship between tenure and monthly income

Highlighted age groups most vulnerable to attrition

Compared gender-based attrition patterns

Provided interactive filtering for deeper analysis
